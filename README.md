# SplitAuth 🔐

A modern and stylish split-screen authentication UI built with HTML, CSS, and JavaScript.
SplitAuth features smooth animations, responsive design, OTP verification flow, premium UI effects, and a clean user experience for login and signup systems.

🌐 **Live Demo:**
[SplitAuth Live Website](https://shankar2006458.github.io/SplitAuth/?utm_source=chatgpt.com)

---

## ✨ Features

* Modern split-screen authentication layout
* Smooth animations and transitions
* Responsive mobile-friendly design
* OTP verification UI
* Interactive buttons and hover effects
* Glassmorphism inspired cards
* Clean and premium interface
* Dark aesthetic theme
* Fully frontend based
* Easy to customize
